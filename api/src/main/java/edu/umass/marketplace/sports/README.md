# Sports Module

This module will handle sports coordination features for Everything UMass.

## Planned Features
- Pickup game organization
- Team finding
- Sports event scheduling
- Player matching

## Structure
- `controller/` - REST API endpoints
- `service/` - Business logic
- `repository/` - Data access layer
- `model/` - Entity classes
- `dto/` - Data transfer objects
