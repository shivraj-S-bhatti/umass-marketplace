# Sports Module (Frontend)

This module will handle sports coordination features for Everything UMass.

## Planned Features
- Pickup game organization
- Team finding interface
- Sports event scheduling
- Player matching UI

## Structure
- `components/` - React components specific to sports
- `pages/` - Sports-related pages
- `hooks/` - Custom React hooks for sports features
- `api/` - API client for sports endpoints
- `types/` - TypeScript types for sports
