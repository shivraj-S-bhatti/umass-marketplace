# Events Module (Frontend)

This module will handle event organization and discovery for Everything UMass.

## Planned Features
- Event creation and management UI
- Event discovery and search
- RSVP functionality
- Event calendar interface

## Structure
- `components/` - React components specific to events
- `pages/` - Event-related pages
- `hooks/` - Custom React hooks for events
- `api/` - API client for event endpoints
- `types/` - TypeScript types for events
