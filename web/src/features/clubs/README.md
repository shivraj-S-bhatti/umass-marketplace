# Clubs Module (Frontend)

This module will handle student club management for Everything UMass.

## Planned Features
- Club creation and management UI
- Club member management interface
- Club event organization
- Club discovery and search

## Structure
- `components/` - React components specific to clubs
- `pages/` - Club-related pages
- `hooks/` - Custom React hooks for clubs
- `api/` - API client for club endpoints
- `types/` - TypeScript types for clubs
