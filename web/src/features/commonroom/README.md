# Common Room Module (Frontend)

This module will handle community forums and discussions for Everything UMass.

## Planned Features
- Forum threads and posts UI
- Discussion categories
- User reputation display
- Moderation interface

## Structure
- `components/` - React components specific to forums
- `pages/` - Forum-related pages
- `hooks/` - Custom React hooks for forums
- `api/` - API client for forum endpoints
- `types/` - TypeScript types for forums
